Accessibility Statement

TRIZEL is committed to ensuring digital accessibility for all people, including those with disabilities. We are continually improving the user experience for everyone and applying the relevant accessibility standards.

Conformance Status

This website aims to conform with Web Content Accessibility Guidelines (WCAG) 2.2 at the AA level. WCAG 2.2 is aligned with ISO/IEC 40500:2025, the international standard for web accessibility.

"AA conformance" means that our content meets all Level A and Level AA Success Criteria, or a conforming alternate version is provided.

Accessibility Features

This website includes the following accessibility features:

  • Keyboard Navigation: All interactive elements are accessible via keyboard
  • Skip Links: "Skip to main content" links at the top of each page
  • Focus Indicators: Visible focus indicators for keyboard navigation
  • Semantic HTML: Proper use of HTML5 semantic elements and ARIA landmarks
  • Multilingual Support: Content available in English, French, German, Russian, Chinese (Simplified), and Arabic
  • RTL Support: Proper right-to-left text direction for Arabic content
  • Touch Target Sizing: Minimum 24Ã—24 pixel touch targets (WCAG 2.2 Level AA)
  • Reduced Motion: Respects user preferences for reduced motion
  • High Contrast: Enhanced contrast support for users with low vision
  • No JavaScript Required: Core functionality works without JavaScript
  • No External Dependencies: All resources served locally (no CDNs, no tracking)

Standards & Guidelines

This website has been designed and developed with reference to:

  • Web Content Accessibility Guidelines (WCAG) 2.2 Level AA
  • ISO/IEC 40500:2025 (Information technology â€” W3C Web Content Accessibility Guidelines)

Technical Specifications

Accessibility of this website relies on the following technologies:

  • HTML5
  • CSS3
  • WAI-ARIA 1.2
  • SVG (for icons, with appropriate aria-hidden attributes)

These technologies are relied upon for conformance with the accessibility standards used.

Assessment Approach

TRIZEL assessed the accessibility of this website through the following methods:

  • Self-evaluation using automated and manual testing tools
  • Keyboard-only navigation testing
  • Screen reader compatibility testing
  • Color contrast analysis
  • Multilingual content verification

Known Limitations

Despite our best efforts to ensure accessibility, there may be some limitations. If you encounter any accessibility barriers, please contact us (see below).

Archive Note: Content in the /artifacts/ directory represents frozen, historical research outputs and may have different accessibility characteristics from the main site.

Feedback

We welcome your feedback on the accessibility of the TRIZEL website. If you encounter accessibility barriers or have suggestions for improvement, please contact us:

  • Method: Via GitHub Issues on the appropriate TRIZEL repository
  • Response Time: We aim to respond to accessibility feedback within 5 business days
