Objectif

L'Index Mondial d'Observation et de Production (GOI) est un registre scientifique fournissant des métadonnées statiques et vérifiables pour l'écosystème TRIZEL.

Cet index catalogue :

  • Observations : Sources de données, archives et sorties de surveillance
  • Production : Résultats scientifiques et résultats d'analyse
  • Référentiels : Structure de l'écosystème et inventaire des composants
  • Instantanés : Ensembles de référence figés avec vérification d'intégrité

Sections de l'index

  • Observations

    Registre des sources de données d'observation et des métadonnées

  • Production

    Registre des résultats de production scientifique

  • Référentiels

    Inventaire et structure de l'écosystème

  • Instantanés

    Registre des instantanés publiés

Comment vérifier

Toutes les données GOI sont statiques et vérifiables :

  1. Source de données statique : /goi/data/global-index.json
  2. Historique des commits : Toutes les mises à jour sont suivies via les commits Git dans ce dépôt
  3. Validation de schéma : Toutes les données sont conformes aux schémas JSON publiés dans /goi/schemas/
  4. Aucune exécution à l'exécution : Pas de JavaScript, pas d'API, pas de récupération de données dynamique
