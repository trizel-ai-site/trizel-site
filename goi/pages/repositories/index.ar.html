التحقق

جميع إدخالات المستودعات قابلة للتحقق من خلال:

  1. سجل الالتزام Git في هذا المستودع
  2. التحقق من صحة مخطط JSON (المخطط: /goi/schemas/repository.meta.schema.json)
  3. مراجع متقاطعة لعناوين URL الفعلية لمستودع GitHub
  4. التحقق من صحة طبولوجيا الطبقة مقابل بنية النظام البيئي
